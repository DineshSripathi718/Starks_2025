Variables

Variables are used to hold the data temp in memory.

Rules :

  • We shouldn't start variable name with numbers
  • We shouldn't use special characters except _ and $ when declaring variables
  • We shouldn't use the keywords as variable names

Declaration

  • Let : (ES6)
    we can assign/ change value to the variable at any point during execution.
  • const : (ES6)
    we can assign value at the time of declaring the variable and we can't override the value.
  • var :
    we can assign/ change value to the variable at any point during execution.

DataTypes :

Used to define the type of data we are storing inside the variable

List of datatypes

  • numbers - standard numbers and decimal
  • string - collection of alphabets and special characters
  • boolean - true or false
Javascript it is a dynamically typed lang. It will automatically assign datatype to the variable based on the store value inside it

Operators :

Tools -> to perform an operation.

  1. Assignment ops (=) :
    used to assign values to the variable.
    1. assignment :
      let variableName = value;
    2. shorthand assignment :
      we use shorthand assignment when we are using a certain variable for the math operation and want to store the result inside the same variable then we use this shorthand assignment operator 
                                          
                                        let value1 = 6;
                                        let value2 = 7;

                                        //add value2 with value1 and store the result inside value1

                                        console.log(value1+" before the operation")

                                        value1 += value2; //value1 = value1 + value2

                                        console.log(value1+" after the operation")
  2. Arth Ops (+, -, *, / , %, **) :
    To perform mathmatical operations we use the arth ops.

    • addition : 

                                          let value1 = 4;
                                    let value2 = 5;
                                    let result = value1 + value2;

                                    console.log(result);

    • subtraction : 

                                          let value1 = 4;
                                    let value2 = 5;
                                    let result = value1 - value2;

                                    console.log(result);

    • multiplication : 

                                          let value1 = 4;
                                    let value2 = 5;
                                    let result = value1 * value2;

                                    console.log(result);

    • division : 

                                          let value1 = 4;
                                    let value2 = 5;
                                    let result = value1 / value2;

                                    console.log(result);

    • Modulo Division

      When we are using modulo division it will return the very 1st reminder of the division operation. 

                                          let value1 = 14;
                                    let value2 = 5;
                                    let result = value1 % value2;

                                    console.log(result);

    • Exp : 

                                          let value1 = 5;
                                    let value2 = 3;
                                    let result = value1 ** value2;

                                    console.log(result);
    NOTE : Arth ops follows BODMAS rule. Bracket Of Division Multiplication Addition Substraction
